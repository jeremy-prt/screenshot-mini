## Features

- **Capture** — Full screen, area, window, OCR text extraction
- **Floating preview** — Copy, edit, save, drag & drop, pin, auto-dismiss
- **Editor** — 11 tools: arrows, text, blur, shapes, freehand, numbered steps, backgrounds
- **Backgrounds** — 18 gradients + solid colors, padding, rounded corners, shadow
- **History** — Thumbnail grid, re-edit, copy, drag & drop
- **Auto-update** — Built-in Sparkle updater
- **Multilingual** — French, English, Spanish, German
- **Themes** — System / Light / Dark

## Install

Download the `.dmg` from [Releases](https://github.com/jeremy-prt/orby/releases), or see the [install guide](https://jeremy-prt.github.io/orby/setup).

Requires **macOS 15 (Sequoia)** or later.

> **Note:** The app is not signed with an Apple Developer certificate. As a student, I can't afford the $99/year fee. macOS will ask you to allow the app on first launch (System Settings → Privacy & Security → Open Anyway). The entire source code is available here for review.

## Build from source

```bash
git clone git@github.com:jeremy-prt/orby.git
cd orby
bash build-app.sh
```

Requires macOS 15+ and Swift 6.2.
